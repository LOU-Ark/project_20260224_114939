Our Paradigm

「知能と感情の交差点に、新しい学問の座標を置く」

我々の目的は、単なる自動化の推進ではありません。ロボット心理学という未知の領域を定義し、社会的な価値を提示すること。 「便利」の先にある「心地よさ」を数値化し、2030年の人間社会に不可欠なインフラとしての信頼を構築します。

FRAMEWORK

定量と定性を統合する評価指標
